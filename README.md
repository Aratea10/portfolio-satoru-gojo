Portfolio web responsive inspirado en **Satoru Gojō** de Jujutsu Kaisen, desarrollado con **HTML5** y **CSS3 puro** como práctica del Bootcamp de Desarrollo Web FullStack en [KeepCoding](https://keepcoding.io/).

---

## ✨ Características principales

### 🎨 Diseño y UX

- **Diseño Mobile First** — Optimizado para móviles primero
- **Tema oscuro coherente** — Inspirado en la estética de Jujutsu Kaisen
- **Animaciones CSS puras** — Efectos visuales sin JavaScript
- **Navegación intuitiva** — Menú burger responsive con CSS puro

### ⚡ Funcionalidades técnicas

- **Validación de formularios HTML5** — Campos requeridos y patrones regex
- **Grid y Flexbox** — Layouts responsivos modernos
- **Variables CSS** — Mantenibilidad del código
- **Semántica HTML5** — Mejor SEO y accesibilidad
- **Páginas de error** — 404 y 500 personalizadas

### 📱 Responsive Design

- **Móvil**: < 768px (enfoque principal)
- **Tablet**: ≥ 768px
- **Desktop**: ≥ 1024px

---

## 🔧 Tecnologías utilizadas

- **HTML5** — Estructura semántica
- **CSS3** — Grid, Flexbox, animaciones con @keyframes, variables CSS y media queries

## 🚀 Cómo ejecutar

### Método 1: Abrir directamente

Descargar el proyecto y abrir `index.html` en el navegador.

### Método 2: Servidor local (recomendado)

Con Python:
````bash
python -m http.server 8000
````

Con Node.js:
````bash
npx http-server
````
